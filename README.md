# pochi
jobcanに出退勤を入力するツール

## Usage

```bash
pochi.rb -t start # start on current time
pochi.rb -t start -d '10:00' # start on today at 10:00
pochi.rb -t start -d '01-10 10:00' # start on current year at "01-10 10:00"
pochi.rb -t finish # finish on current time
pochi.rb -t finish -d '19:00' # finish on today at 19:00
pochi.rb -t finish -d '01-10 19:00' # start on current year at "01-10 19:00"
```
